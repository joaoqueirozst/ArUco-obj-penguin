# Detection of ArUcos markers with AR

</h1>

This work was developed in the Computer Vision course ([CV](https://opencv.org/)), an elective course offered by the institution, in which the concepts seen in class about ArUcos were applied to [ArUcos marker detection](https://docs.opencv.org/4.x/d5/dae/tutorial_aruco_detection.html) with images and augmented reality.
<br>

ArUco markers are markers widely used in `Computer Vision` and `Augmented Reality` applications. Each marker contains a unique identification pattern that allows algorithms to detect its position and orientation in real time through a camera. Using these markers, project virtual objects, perform camera tracking, and create interactive Augmented Reality environments with high precision and low computational cost.
